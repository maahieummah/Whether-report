# Whether-report
This Python script fetches real-time weather information for any city using the OpenWeatherMap API. It takes a city name as user input, sends an HTTP request to the API, and displays key weather details such as temperature, feels-like temperature, humidity, and weather condition in a readable format.
The program also includes basic error handling to gracefully handle invalid city names, API errors, or unexpected responses.

Key Features:

Uses requests to make API calls

Supports temperature in Celsius

Displays real-time weather data

Handles API errors with meaningful messages

Tech Stack: Python, REST API, OpenWeatherMap
